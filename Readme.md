

This project demonstrates the use of basic HTML and CSS selectors, including element, class, ID, and descendant selectors.



- Styled headings using class selectors
- Applied a lime background using an ID selector
- Changed the color of specific text using descendant selectors
- Styled paragraphs using element selectors
- Customized a paragraph using an ID selector
- Styled text inside a paragraph using a span selector
- Created unordered and ordered lists
- Styled hyperlinks without using class or ID on the `<a>` tag



- HTML5
- CSS3
